# 2026 전교인 리트릿 — 관리자 사용법

> 💡 **30초 요약**
> 요약 탭의 **‘오늘 할 일’**을 위에서부터 누르면 그 탭으로 갑니다. 그대로 따라가면 끝나요.
> 순서: **① 그룹정리 → ② 같은 방 요청 → ③ 방배정 → ④ 입금·확인**
> 화면이 헷갈리면 각 탭의 **‘ⓘ 이 탭 사용법’**, 전체는 헤더 **‘📖 사용법’**을 누르세요.

---

## 1. 접속하기

1. 등록 사이트 주소 끝에 **`#admin`** 을 붙여 접속합니다.
2. **PIN** 입력 → 접속. (PIN은 담당자에게 문의)
3. 데이터가 안 보이면 우측 위 **[새로고침]**.

> ⚠️ 시트가 자동 정리(enrich)된 뒤의 값이 보입니다. 새 신청이 많이 들어왔는데 안 보이면 담당자에게 “시트 정리 돌려달라”고 요청하세요.

---

## 2. 꼭 알아야 할 개념 2가지

이거 두 개만 구분하면 90%는 이해됩니다.

| 개념 | 뜻 | 어디서 다루나 | 비용 영향 |
| --- | --- | --- | --- |
| **그룹** | **비용**을 함께 내는 단위 (가족 등) | **그룹정리** 탭 | 합치면 **객실 그룹가 등 비용이 다시 계산됨** |
| **방** | **같이 자는** 단위 (숙소 배정) | **같은 방 요청 / 방배정** 탭 | 비용과 무관, **방 번호만** 같이 붙음 |

> 📌 핵심: **“돈도 같이” = 그룹정리에서 합치기**, **“방만 같이” = 같은 방 요청에서 묶기**.
> 같은 방을 쓴다고 비용이 합쳐지는 게 아닙니다 (각자 냅니다).

---

## 3. 작업 순서 (이 순서대로면 됩니다)

| 단계 | 탭 | 하는 일 |
| --- | --- | --- |
| ① | **그룹정리** | 잘못 묶이거나 떨어진 가족·그룹을 바로잡기 |
| ② | **같은 방 요청** | “○○랑 같은 방” 요청을 모아 방으로 묶기 (비용 각자) |
| ③ | **방배정** | 교회가 정해줄 사람을 방에 배치 (자동배치 + 드래그) |
| ④ | **입금·확인** | 입금 확인하고, 미제출자 연락 |

> ✅ 요약 탭 **‘오늘 할 일’** 체크리스트가 각 단계의 남은 건수를 보여주고, 누르면 해당 탭으로 이동합니다. 0건이 되면 ✅완료로 바뀝니다.

## 4. 탭별 한 줄 요약

| 탭 | 한 줄 설명 |
| --- | --- |
| **요약** | 전체 현황 + ‘오늘 할 일’. 여기서 시작 |
| **그룹정리** | 비용 단위(그룹) 합치기/나누기. 방 일괄 맞추기 |
| **같은 방 요청** | “같은 방 해주세요” 요청 자동 묶기 (비용 각자) |
| **방배정** | 교회 배정 대상자를 방에 배치 (드래그/자동) |
| **입금·확인** | 미입금 확인, 미제출자 연락 |
| **문의** | 성도 문의 자동 분류 (입금/방배정/기타) |
| **버스** | 캠퍼스별 버스 신청자 명단 |

## 5. 자주 하는 작업 Top 5

### ① 잘못 나뉜 가족 합치기
1. **그룹정리** 탭으로.
2. 위쪽 **‘⚠ 확인이 필요해요(의심 목록)’** 에서 같은 번호로 갈라진 그룹을 찾고 **[이 그룹들 합치기]**.
3. 목록에 없으면, 아래 **‘전체 그룹 목록’**에서 합칠 그룹들을 체크 → **[선택 N그룹 합치기]**.
4. 확인창에서 대상 명단과 “비용 재계산됨”을 보고 **[합치기]**.

### ② 사람을 다른 그룹으로 옮기기 / 혼자로 빼기
1. **그룹정리** → **‘🔎 이름으로 이동’**.
2. 옮길 사람 검색 → 선택 → 보낼 그룹을 이름으로 검색 → 선택.
3. 혼자로 빼려면 **[단독으로 분리]**.
4. 확인창 확인 → 이동. (방 배정도 같이 따라옵니다)

### ③ “같은 방 해주세요” 요청 처리
1. **같은 방 요청** 탭으로.
2. 🟢/🟡 카드에서 멤버와 원문(📝)을 확인.
3. **[이 방 같이 쓰기]** → 같은 방 번호가 붙습니다. (비용은 각자)
4. 🔴는 먼저 풀어야 합니다 (예: 남녀 혼방 → 나눠서, 미등록자 → ‘미제출 추가’).

### ④ 명단엔 있는데 신청 안 한 사람 추가
1. **요약** 탭 → **‘📋 미제출 추정 명단’** 펼치기.
2. 실제 사람이 맞으면 **[미제출 추가]** (확인창 뜸).
3. 이름이 아니거나 이미 신청한 사람이면 **[제외]**.

### ⑤ 입금 확인
1. **입금·확인** 탭 → **미입금** 목록.
2. 입금된 사람 체크 → **[선택 N명 입금확인]** → 확인창 → 처리.

## 6. 이럴 땐 이렇게 (자주 막히는 상황)

| 상황 | 이렇게 |
| --- | --- |
| **이름이 똑같은 사람(동명이인)** | 시스템이 A/B로 구분해 둡니다(예: 이사랑A·이사랑B). 같은 방/그룹 요청에서 ‘동명이인’ 경고가 뜨면 누구인지 확인 후 진행. |
| **‘같은 방 요청’에 이상한 단어가 이름으로 뜸** | 문장 일부가 잘못 잡힌 것. 옆 **[제외]** 버튼으로 치우면 다시 안 보입니다. |
| **합쳤는데/옮겼는데 잘못함** | 그룹정리에서 다시 나누거나 옮겨 되돌립니다. (완전 자동 되돌리기는 없으니 신중히) |
| **방이 그룹과 안 맞음** | 그룹정리 → **‘🛏️ 그룹 기준으로 방 맞추기’** 로 한 번에 통일. |
| **설악산뷰가 몇 방 필요한지** | 요약 → **‘🏔️ 설악산뷰 방 현황’**. 인원보다 **방 수**를 봅니다. |
| **숫자(미제출 추정 등)가 좀 안 맞음** | 자유 텍스트에서 뽑은 **어림수**라 오차가 있습니다. 그대로 믿지 말고 명단으로 확인. |

---

## 7. 신호 색 & 안전장치

- 🔴 **먼저 풀어야 함** (남녀 혼방·정원 초과·이미 다른 그룹 등) — 버튼이 잠깁니다.
- 🟡 **확인 후 진행** (동명이인·미등록자 등).
- 🟢 / ✅ **바로 가능 / 이미 완료**.
- **합치기·이동·방 맞추기·일괄 저장·입금확인**은 누르면 **확인창**이 뜹니다. “**무엇이 어떻게 바뀌는지 / 비용이 바뀌는지**”를 보고 진행하세요.

---

## 8. 담당자(임성현)에게 연락해야 할 때

- 💸 **비용이 이상하게 계산됨** (금액이 안 맞는 그룹)
- ⚠️ **빨간 오류 메시지**가 뜸 / 저장이 안 됨
- 🔄 화면이 안 바뀜 (배포·시트 정리 문제로 추정)
- 🧮 신청이 많이 들어왔는데 관리자 화면에 안 보임 → 시트 정리(enrich) 요청

> 그 외 자동 분류(같은 방 요청·문의·미제출 명단)는 **어림짐작**이라 가끔 틀립니다. 사람이 한 번 확인하고 진행하면 됩니다.
