# 2026 전교인 리트릿 — 관리자 사용법

> 💡 **핵심 2가지** — **그룹** = 비용 함께 내는 단위(가족) / **방** = 같이 자는 단위. “돈도 같이”는 **그룹정리**, “방만 같이”는 **같은 방 요청**.
> **순서: 그룹정리 → 같은 방 요청 → 방배정 → 입금·확인**

---

## 탭별 설명

| 탭 | 무엇을 하나 |
| --- | --- |
| **요약** | 전체 현황 + **‘확인 리스트’**(남은 할 일). 여기서 시작하고, 항목을 누르면 그 탭으로 이동. |
| **그룹정리** | 비용을 함께 내는 **그룹**을 바로잡기 — 합치기 / 이동 / 분리. *(합치면 비용이 다시 계산됨)* |
| **같은 방 요청** | “○○랑 같은 방” 요청을 모아 **방으로 묶기** *(비용은 각자)*. 빈자리는 조건 맞는 사람 추천. |
| **방배정** | 교회가 정해줄 사람을 **방에 배치** — 자동 배치 + 드래그. ‘그룹대로 방 맞추기’도 여기서. |
| **입금·확인** | 미입금 확인, 명단엔 있는데 신청 안 한 사람 연락. |
| **문의** | 성도 문의를 입금/방배정/기타로 **자동 분류**. |
| **버스** | 캠퍼스별 버스 신청자 명단. |

> 👉 각 탭 위의 **‘ⓘ 이 탭 사용법’** 을 누르면 그 탭 사용법이 자세히 나옵니다.

---

## 알아둘 것

- **신호색**: 🔴 먼저 풀기 · 🟡 확인 후 · 🟢/✅ 바로 가능·완료
- **합치기·이동·방 맞추기·저장**은 누르면 **확인창**이 떠요 (비용 바뀌는지 보고 진행).
- 자동 분류(같은 방 요청·문의·미제출 명단)는 **어림짐작**이라 가끔 틀려요 → 사람이 확인하고 진행.
- **비용이 이상 / 오류 / 화면 안 바뀜** → **웹앱 제작자**에게 문의.
